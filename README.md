# pitching_analysis_app

単眼の野球投球フォーム動画に対して、代表フレームで指定したマウンド平面4点からホモグラフィを計算し、平面が正面向きになるよう補正した映像をブラウザ上で再生する Web アプリです。

## 使い方

1. `index.html` をブラウザで開く。
2. 動画ファイルを読み込む。
3. 身長(cm)を入力する。
4. 代表フレームで停止し、以下を設定する。
   - 平面P0(左上)
   - 平面P1(右上)
   - 平面P2(右下)
   - 平面P3(左下)
   - 頭頂
   - 足元
5. 表示モード（自動トリミング / 黒余白維持）を選択する。
6. 「補正実行」→「再生」で補正後映像を確認する。
7. 必要に応じて「WebM保存（任意）」で保存する。

## 構成

- `index.html`: UI レイアウト
- `style.css`: スタイル
- `js/state.js`: 状態モデルと初期化/リセット
- `js/dom.js`: DOM参照の取得
- `js/transform.js`: スケール計算とホモグラフィ計算
- `js/overlay.js`: 基準点・軸のオーバーレイ描画
- `js/corrected.js`: 補正フレーム描画・再生・保存
- `js/main.js`: イベント配線とアプリ全体の制御
